# Web開発のすすめ

### KCS Web班　骨なしチキン

KCS :: Keio Computer Society は、慶應義塾大学で唯一、そして日本最古のコンピュータサークルです。「パソコン使って全部やる」をモットーに、Web・AI・System（低レイヤー）・競プロ・ゲーム・音楽・CG の７班に分かれて活動しています。

本記事は、2026年度 Web班の講習会資料です。Web開発（フロントエンド＋バックエンド）の基礎知識を体系的にまとめました。
